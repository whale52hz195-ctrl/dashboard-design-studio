# Alkasser Dashboard (Tingle)

## Overview
Admin dashboard for managing a social media application called "Tingle". Built with React, TypeScript, Vite, and Firebase. Uses Tailwind CSS with shadcn/ui components.

## Recent Changes
- 2026-02-16: Initial Replit setup — configured Vite for port 5000, installed dependencies via npm, set up static deployment.

## Project Architecture
- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS + shadcn/ui (Radix primitives)
- **Backend**: Firebase (Firestore, Storage)
- **Routing**: React Router v6
- **State**: TanStack React Query
- **Structure**:
  - `src/pages/` — Page components (Login, Dashboard, UserManagement, etc.)
  - `src/components/` — Shared UI components
  - `src/data/` — Mock data files
  - `src/hooks/` — Custom React hooks
  - `src/lib/` — Firebase config and utilities
  - `src/storage/` — Static assets
  - `public/` — Public static files
